# 🚀 GoalTrail – Progress Tracker App

GoalTrail is a modern and interactive **personal progress tracker** designed to help users manage tasks, monitor productivity, and stay motivated. Built using React, this application provides a clean and engaging interface for organizing daily goals and tracking progress efficiently.

## ✨ Features

- ✅ **Task Management** – Add, update, complete, and delete tasks easily  
- 📊 **Progress Visualization** – View completion stats with dynamic progress charts  
- 📝 **Notes Section** – Save important thoughts and ideas instantly  
- ⏰ **Reminders** – Set time-based reminders for important activities  
- 👤 **User Profile** – Manage personal details and track overall productivity  
- 🔐 **Authentication System** – Simple login, registration, and password management  
- 🎯 **Category & Priority Filtering** – Organize tasks based on categories and priorities  
- 💡 **Motivational Quotes** – Daily inspiration to keep you going  

## 🛠️ Tech Stack

- **Frontend:** React (Functional Components & Hooks)  
- **Styling:** Inline CSS with modern UI design  
- **State Management:** React useState & useEffect  

## 🎯 Purpose

The goal of this project is to provide a **simple yet powerful productivity tool** that encourages users to stay consistent with their goals. It promotes better time management, self-discipline, and personal growth through structured tracking.
